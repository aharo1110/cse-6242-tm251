# CSE6242 Team 251
## Sentiment Analysis from 2018 FIFA World Cup Tweets

To run this project create a Python virtual environment (version 3.12 recommended) with `flask` and `flask-bootstrap` installed.    
Then, run `flask run` in your virtual environment.

### How to use

If you wish to view a matchdate, select from the available options. Once there you can see interactions over time in the data.    
Any detected event will show, and upon hovering will display information about the real-time sentiment surrounding that event. The different methods tested are shown with results.
Additionally, a network graph will show with individual nodes corresponding to accounts. Hovering over them will show information about them.

If you wish to view the set of events, click on the button to go to that page. It will then show the set of known events and information relating to those events, similar to the matchdate page.
